# 更新日志 / Changelog

## [v2.2.0] - 2026-03-26

### 🆕 新增功能

- **`predict_v4.py`** - 时间加权优化版预测
  - 时间加权 - 近期数据权重更高 (RECENT_WEIGHT=0.6)
  - 多周期分析 - 短期50期/中期200期/长期1000期
  - 遗漏值动态权重 - 冷号回补概率建模
  - 自适应权重 - 基于历史验证调优

- **`fetch_history_v3.py`** - 数据获取v3
  - 增量更新支持
  - 自动化数据同步

- **`fix_csv.py`** - CSV修复工具
  - 编码修复
  - 数据清洗

- **NiceGUI Web界面** - Web预测系统
  - 顶部菜单导航
  - 图表瀑布流展示
  - 历史数据颜色标识（重复/连号/冷号）
  - 自动化数据更新

### 🔧 改进

- 核心算法优化
  - 核密度估计(KDE) 30%
  - Beta分布 25%
  - 三峰高斯混合 25%
  - 历史频率统计 20%

- 胆拖策略优化
  - 红球: 4胆码 + 3热拖 + 2冷拖
  - 蓝球: 1胆码 + 2拖码

### 📊 数据更新

- 历史数据持续更新

## [v2.1.0] - 2026-03-25

### 🆕 新增功能

- **`fetch_all_history.py`** - 全量历史数据爬虫
  - 从 500star.com 抓取完整历史开奖数据
  - 支持增量更新（断点续爬）
  - 自动去重合并

- **`predict.py`** - 一键预测脚本
  - 整合所有分析模块
  - 图算法增强预测（PageRank + 共现网络）
  - 自动生成预测报告

- **`install.sh`** - VPS 一键部署脚本
  - 自动安装 MariaDB + Python 依赖
  - 配置数据库和定时任务

### 🔧 改进

- **`graph_predictor.py`** - 图算法核心模块
  - PageRank 算法分析号码核心度
  - 号码共现网络构建
  - 区域分布聚类分析
  - 间隔规律挖掘

- **`lottery_spider_v2.py`** - 多源爬虫
  - 新浪财经 + 腾讯彩票双源
  - 备用网页解析
  - CSV 中文列名修复

- **`config.py`** - 配置重构
  - 支持环境变量覆盖
  - 数据库密码可配置

### 📊 数据更新

- 历史数据从 758 条扩展至 **2225 条**
- 时间跨度：2011年6月 ~ 2026年3月
- 覆盖 2011~2026 完整年度

### 🐛 修复

- CSV 中文列名兼容问题（`期号` → `period` 等）
- 间隔规律计算 bug
- MySQL 数据库初始化脚本

## 技术架构

```
lottery_data.csv          ← 主数据文件
     ↓
predict.py                ← 预测入口
     ↓
├── LotteryAnalyzer       ← 核心分析器
│   ├── 频率统计           ← 历史出现频次
│   ├── PageRank          ← 号码核心度
│   ├── 共现网络           ← 号码关联关系
│   └── 区域分析           ← 号码分区统计
│
├── 图算法增强             ← 网络分析
│   ├── 节点重要性排名
│   └── 聚类热区发现
│
└── 蒙特卡洛采样           ← 概率预测
    └── 分层采样策略
```
