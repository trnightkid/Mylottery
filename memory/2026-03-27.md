## 凌晨工作记录

### 彩票分析项目 - v2 GAN 模型

**问题诊断（断线原因）：**
- v1 模型存在严重 **mode collapse**：红球只循环 7 个固定号码，蓝球只循环 5 个
- v1 判别器过强（学习率与生成器相同），导致生成器被压制
- v1 使用 `torch.topk` 输出导致模型锁定在最高概率的几个号码

**v2 改进：**
- Softmax + 贪心采样替代 topk
- G:D 训练步数比 = 2:1（生成器训练更频繁）
- G 学习率 2e-4，D 学习率 5e-5（判别器更慢）
- 多样性损失（entropy-based）+ 特征匹配损失
- 蓝球判别器也独立训练
- 红球覆盖从 7/33 → 21/33（显著改善）

**v2 回测结果（200期）：**
- 红球平均命中：1.11个（随机基准约1.08个）
- 红球3+命中率：5.5%
- 蓝球命中率：8.0%
- **结论：和随机猜测几乎没有差别**

**根本原因：** 双色球是完全随机游戏，历史数据无规律可循。GAN 学到的是训练集频率分布（回归均值），不是预测能力。
